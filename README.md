# CS2 Weapon Paints Web
A simple website to use [cs2-WeaponPaints](https://github.com/Nereziel/cs2-WeaponPaints/) plugin.

## Features
- Knifes
- Skins
- Gloves
- Agents
- Music Kits  

**You can set every skins on every weapons, such as "P90 Fade", "AK-47 Gamma Doppler".**
